Getting and Cleaning Data Course Project
================================================================================

Description
================================================================================
The R script `run_analysis.R` is used to clean and tidy the Human Activity Recognition dataset.

Files
================================================================================
- run_analysis.R: Performs data cleaning and merging to create tidy_data_avg.txt 
- CodeBook.md: Describes variables and transformations
- tidy_data_avg.txt: Final tidy dataset

How to Run
================================================================================
1. Download and unzip the dataset "UCI HAR Dataset"
2. Set your working directory in RStudio
3. Run the script `run_analysis.R`
